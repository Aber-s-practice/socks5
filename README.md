# socks5

A little Socks5 Server by ThreadServer.

#### Just Use

You will get a socks5 server (non-auth)

```bash
python3 Socks5.py
```

#### Self-customization

Can read [DefaultSession](https://github.com/AberSheeran/Socks5/blob/master/Socks5.py#L223) in Socks5.py
